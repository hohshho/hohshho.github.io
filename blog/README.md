# hohshho Blog
 - https://github.com/junhobaik/junhobaik.github.io 테마 적용해 만들었습니다.

**Built with:**

- Gatsby
- React
- Redux
- TypeScript
- SASS (SCSS)
- Typography.js
- Google Fonts
- Google Analytics
- Google AdSense
- ...

**기능**

- 선(Border)이 없는, 색이 최소화된 심플한 디자인
- 마크다운 게시물 작성 (Markdown + emoji, ktex)
- 검색 엔진 최적화 (SEO), Chrome DevTools Audits SEO 100/100
- 사파리 브라우저 등 게시물내 읽기 모드(Reader Mode) 지원
- Google Analytics 지원
- Google Adsense 지원
- Disqus 댓글 기능 지원
