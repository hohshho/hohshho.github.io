# 1. SQLite .명령어 정리
> SQLITE의 특징은 명령어 앞에 마침표(.)를 붙인다.

파일 접속
```bash
$ sqlite3 {파일명}
```

파일 접속 종료
```bash
> .quit
> .exit
```

테이블 확인
```bash
> .tables
```

테이블 구조 확인
```bash
> .schema access_log
```

헤더(컬럼 명) 함께 출력
```bash
> .header on
```

라인별로 나눠 출력
```bash
> .mode line
```

컬럼별로 나눠 출력
```bash
> .mode column
```

추가 기능 확인
```bash
> .help
```

# 2. 파일 I/0
> sqlite3는 쿼리 결과를 표준 출력으로 보낸다.
 - ".output", ".once" 명령어를 사용하여 변경 가능


1) .OUTPUT 명령어 
 > .output에 인수로 파일명 넣으면 해당 파일에 기록 (.output을 사용하면 표준 출력 상태로 되돌아간다)

2) .import 명령어를 사용해 csv데이터를 sqlite 테이블로 가져올 수 있다.
```bash
sqlite> .import C:/CVS/aaa.csv aaa
```

**주의할 점**
 - SQLite3은 기본적으로 UTF-8형식 / 윈도우 엑셀은 CP949형식
