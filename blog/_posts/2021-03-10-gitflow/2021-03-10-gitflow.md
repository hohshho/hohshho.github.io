### Git Branch 생성 및 checkout

```bash
$ git checkout -b 
```

### Git Branch 삭제

```bash
$ git branch -d {브런치 명}
```

### Git Branch 확인

```bash
$ git branch

$ git branch -v
```

### Git Branch 상태 확인

```bash
$ git branch --merged
```
 > 이미 Merge한 브랜치 목록을 확인한다.

### Git log 확인

```bash
$ git log --oneline --graph
```
